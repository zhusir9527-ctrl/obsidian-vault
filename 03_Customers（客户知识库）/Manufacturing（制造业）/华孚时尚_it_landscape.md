# 华孚时尚 · IT 现状

## 现有架构

| 类别 | 现状 | 问题 |
|------|------|------|
| 基础设施 | 传统制造IT：物理服务器为主 | 扩容慢、运维成本高 |
| 虚拟化 | 推测VMware | license成本可能上涨 |
| GPU资源 | 零散采购，多工厂各自管理 | 利用率<30%，无统一调度 |
| ERP | 用友/金蝶 | 与AI系统未打通 |
| MES | 工厂级MES | 各基地独立部署 |
| 供应链 | 自建供应链平台 | 预测依赖Excel+经验 |

## 核心痛点

1. AI算力底座缺失 — GPU分散各工厂，利用率<30%
2. 色纺配色效率低 — 人工配色，一次成功率~60%，打样周期3-5天
3. 纱线质检以人工为主 — 疲劳导致漏检
4. 供应链预测不准 — Excel+经验判断
5. 多基地算力割裂 — 各厂独立采购，无法共享

## AI智算需求

| 场景 | 描述 | 青云方案 |
|------|------|----------|
| AI配色 | CV+色度模型，替代人工配色 | GPU池化 + AI智算平台 |
| 智能质检 | CV缺陷检测 | GPU推理 |
| 供应链预测 | 需求预测+库存优化 | 数据中台 |
| 智算中心 | 新疆算力底座 | K8s+GPU池化+多厂统一 |

当前青云与华孚时尚正在进行AI智算平台PoC，聚焦纺织品AI配色场景。
