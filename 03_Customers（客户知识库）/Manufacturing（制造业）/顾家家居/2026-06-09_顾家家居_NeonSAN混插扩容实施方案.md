# 顾家家居 NeonSAN 混插扩容实施方案

> 日期：2026-06-09 | 版本：NeonSAN-installer 2.0.17.2 / NeonSAN 3.6.1

## 背景

客户 HDD0 资源池可用空间仅剩约 4TB，且机器较老。若有一台宕机，整个 HDD0 资源池将达 100% 使用率，所有业务受影响。

## 扩容范围

- **资源池**：HDD0（混插）
- **扩容节点**：2 台 DELL R740 混插存储节点
- **主机名/IP**：
  - kuka1r10n02 / 192.168.250.102
  - kuka1r10n03 / 192.168.250.103

## 硬件配置

| 部件 | 规格 |
|------|------|
| CPU | Intel Xeon Silver 4314 @ 2.40GHz ×2 |
| 内存 | 64G ×2 = 128G |
| 系统盘 | SAS_600G ×2 |
| 缓存盘 | NVME 1.92T |
| 数据盘 | SATA_1.2T ×22 |
| 网络 | 双口10Gb（含光模块）×2 |
| OS | QingCloud-20.04.1-AutoInstall |

## 关键注意事项

- 原混插节点已超出最佳数量 12 台（目前 17 台），需先从 default RG 迁移清空一台节点，与本次扩容的 2 台混插节点组成新的 HDD RG
- 扩容与升级/维护互斥，扩容期间终止其他操作
- 操作双人复核，完成功能/性能/HA 测试后交付

## 实施步骤概要

1. 硬件上架 + BIOS 配置 + OS 安装
2. 网络配置（业务 bond + 存储双端口）
3. NeonSAN 环境检查（版本/IO Task/磁盘清理）
4. 收集硬件信息 + 节点初始化
5. 扩容 NeonSAN + 流控配置 + 一致性检查
6. 加入生产 RG + 服务验证
