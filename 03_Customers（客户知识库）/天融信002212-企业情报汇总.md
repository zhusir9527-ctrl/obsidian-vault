## 二、核心产品线

### 2.1 产品矩阵总览

天融信是国内**产品线最全**的安全厂商之一，覆盖网络安全、数据安全、云计算三大领域，号称"100+产品/服务"。

| 产品大类 | 核心产品 | 市场地位 | 备注 |
|----------|----------|----------|------|
| **边界安全** | 防火墙（FW）、下一代防火墙（NGFW）、WAF | 国内TOP 2-3 | 起家产品，连续多年市占率前三 |
| **VPN/加密** | IPSec VPN、SSL VPN、加密机、量子加密 | 国内TOP 2 | 政府/军工加密标配 |
| **入侵检测/防御** | IDS、IPS、APT检测 | 国内TOP 3 | 与防火墙联动形成纵深防御 |
| **安全管理平台** | SOC、态势感知（天融信安全云管平台）、SIEM | 国内TOP 3-5 | 面向toG/toB大型客户 |
| **云安全** | 云防火墙、CWPP、云安全资源池、SASE | 快速追赶 | 与华为云、阿里云等生态合作 |
| **信创安全** | 国产化防火墙、国产化VPN、国产化加密机 | **第一梯队** | 全产品线完成信创适配（飞腾/鲲鹏/海光/龙芯/中标麒麟/UOS） |
| **数据安全** | DLP、数据库审计、数据脱敏、数据分类分级 | 快速增长 | 数据安全法驱动，增长率>40% |
| **端点安全** | EDR、终端安全管理、防病毒 | 中上游 | 通过OEM/自研双路径 |
| **工业互联网安全** | 工控防火墙、工控审计、工业态势感知 | 重点布局 | 能源/制造/烟草等 |
| **车联网安全** | V2X安全、车载防火墙、车联网安全检测 | 前沿布局 | 2022年起重点投入 |
| **安全服务** | 渗透测试、等保测评、红蓝对抗、安全运营（MSS） | 核心能力 | 高毛利、高粘性业务 |

### 2.2 产品演进路线

```
1995-2005：防火墙/VPN/IDS 三件套（安全1.0）
2005-2015：UTM/NGFW/SOC 平台化（安全2.0）
2015-2020：态势感知/云安全/数据安全（安全3.0）
2020-2025：信创安全/零信任/SASE/AI安全（安全4.0）
```

### 2.3 信创安全优势（核心差异化）

- **全产品线信创适配**：防火墙、VPN、WAF、IDS/IPS、加密机等全线产品完成与国产CPU/OS适配
- **信创目录覆盖率高**：进入多个中央及地方信创采购目录
- **军工资质齐全**：拥有涉密信息系统集成甲级、军队装备承制等顶级资质
- **典型案例**：中央部委信创安全替换、省级政务云信创安全

## 四、企业业务（toB/toG）销售模式

### 4.1 目标客户结构

```
toG（政府/军工/事业单位）：~50-55% 营收占比
  ├── 中央部委（中办/国办/公安部/网信办等）
  ├── 省级政府（各省政务云/大数据局）
  ├── 地市级政府
  ├── 军工/涉密单位
  └── 公检法司/应急/税务等垂管系统

toB（大型企业）：~30-35% 营收占比
  ├── 金融（银行/证券/保险）
  ├── 运营商（移动/联通/电信）
  ├── 能源（电网/石油/煤炭）
  ├── 交通（铁路/民航/港口）
  └── 教育/医疗

SMB（中小企业）：~10-15% 营收占比
  └── 主要通过渠道覆盖
```

### 4.2 销售模式详解

| 模式 | 占比 | 特点 |
|------|------|------|
| **直销（KA直客）** | ~60-70% | 总部销售团队+区域销售直拓大客户；针对政府/金融/运营商/能源等战略客户 |
| **渠道分销** | ~25-30% | 省级总代→地市代理商→集成商；覆盖SMB和地市/县下沉市场 |
| **生态合作/集成** | ~5-10% | 与华为/浪潮/中科曙光等ICT厂商联合投标；与集成商（太极/中软/东软等）项目分包 |

### 4.3 典型销售打法

```
【攻城模式】KA大客户突破：
  1. 总部销售+售前+安服三线协同
  2. 「先试用后采购」：免费部署态势感知探针，数据说话
  3. 「从点到面」：单产品突破→等保整改→整体安全方案
  4. 标杆案例复制：省级经验→地市推广

【守城模式】存量客户深挖：
  1. 维保/续费→版本升级→扩容
  2. 防火墙客户→推VPN/IDS→推SOC/态势感知
  3. 老客户新产品（交叉销售率是核心KPI）

【合围模式】渠道覆盖：
  1. 省级总代承担压货/回款/物流
  2. 地市金牌代理承担客情/交付
  3. 厂商提供售前支持和培训赋能
```

### 4.4 销售组织架构（典型区域）

```
区域总经理
  ├── 销售部
  │   ├── KA一组（政府/军工）
  │   ├── KA二组（金融/运营商）
  │   ├── KA三组（能源/交通/大企业）
  │   └── 渠道部（渠道经理×N）
  ├── 售前技术部（售前工程师×N）
  ├── 安全服务部（安服工程师×N）
  └── 综合管理部（商务/行政/HRBP）
```

## 六、2024-2026年战略方向与增长重点

### 6.1 公司级战略（据年报/投资者交流）

| 战略主题 | 具体内容 |
|----------|----------|
| **信创安全深化** | 从党政信创→行业信创（金融/运营商/能源），从办公系统→核心业务系统安全替换 |
| **数据安全做大** | 抓住数据安全法/个保法红利，数据分类分级/数据出境安全/隐私计算等 |
| **云安全+安全云** | 云安全资源池/SaaS安全服务（MSS/MDR），构建持续性收入 |
| **AI+安全** | 安全GPT/智能SOC/自动化响应（SOAR），AI赋能安全运营降本增效 |
| **渠道下沉** | 从省覆盖→地市覆盖，加大SMB市场渗透率 |
| **安全服务转型** | 从卖产品→卖服务，提高经常性收入占比（目标从20%→35%+） |

### 6.2 增长重点排序（2025-2026）

```
第一增长曲线（存量优势）：
  └── 防火墙/VPN等边界安全 → 维持份额，贡献现金流

第二增长曲线（规模化增长）：
  ├── 信创安全 → 行业信创30%+增速
  ├── 数据安全 → 政策驱动40%+增速
  └── 安全服务/MSS → 订阅式收入30%+增速

第三增长曲线（未来布局）：
  ├── 云原生安全（CWPP/CNAPP）
  ├── 车联网安全
  ├── AI安全（大模型安全评测/防护）
  └── 安全出海（一带一路沿线）
```

## 七、对浙江区域业务策略的启示

### 7.1 打法建议框架

```
「三层打法」模型：

【上层——品牌造势】
  ├── 承办浙江省网络安全周/行业论坛
  ├── 发布浙江区域安全态势年度报告
  └── 与浙江网信办/公安厅/大数据局联合实验室

【中层——客户深耕】
  ├── 政府：抓住信创窗口期，政务云安全全栈替换
  ├── 金融：数据安全合规+信创，蚂蚁/网商生态合作
  ├── 运营商：联合移动/电信做政企安全增值包
  └── 制造：工业互联网安全试点→规模化

【下层——渠道覆盖】
  ├── 每个地市发展1-2家金牌渠道
  ├── 渠道赋能：月度培训+季度联合打单
  └── 中小企业安全SaaS包（云端订阅）
```

### 7.2 团队管理关键动作

```
1. 目标分解：省区目标→行业线→个人（季度滚动+月度复盘）
2. 能力提升：售前+销售每周联合演练（方案讲解+竞品对比）
3. 激励机制：信创/数据安全新产品提成系数上浮20%
4. 人才梯队：校招+社招结合，建立"新人90天成长计划"
5. 协同机制：区域与总部行业部双线汇报，重大项目"铁三角"攻坚
```
