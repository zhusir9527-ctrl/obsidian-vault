# 中间件容器化技术白皮书

> Jenkins · Harbor · Redis · Kafka · RabbitMQ
> 青云科技售前技术团队 | 2026-06

## 覆盖内容

| 中间件 | 部署方式 | 故障场景 | 监控命令 |
|--------|:--:|:--:|:--:|
| Jenkins | Docker + compose | 6种常见故障 | 日志/健康检查 |
| Harbor | 在线安装 + compose | 7种常见故障 | 组件状态/存储/DB |
| Redis | 单机/哨兵/集群 | 7种常见故障 | INFO/SLOWLOG/BIGKEYS |
| Kafka | KRaft + compose + UI | 7种常见故障 | Topic/Consumer管理 |
| RabbitMQ | 单机/集群 + compose | 8种常见故障 | rabbitmqctl/diagnostics |

## 核心部署配置

- Jenkins（LTS JDK17）— Docker in Docker + 清华镜像加速
- Harbor v2.12 — docker-compose 全组件 + Trivy 扫描
- Redis 7.4 — 哨兵模式 + AOF 持久化 + 大 Key 扫描
- Kafka KRaft 模式 — 三节点集群 + Kafka UI 监控
- RabbitMQ 4.0 — 三节点集群 + 镜像队列 HA
