# 朱祚德每日时间规划 V1.0

> 原则：保护上午深度时间、集中碎片化沟通、留好撤退口
> 适用：浙江制造业售前，团队2人，客户多线程

---

## 每日时间块

| 时段 | 做什么 | 不做什么 |
|------|--------|---------|
| **08:30 - 09:00** | 到岗、看日历、定今天「一个主任务」 | 不刷微信、不回邮件 |
| **09:00 - 11:30** | 🔋 高能耗：写合同 / 出方案 / 标书 / 报价 | 不约客户、不接电话、不开会 |
| **11:30 - 12:00** | 微信集中回复、快速协调 | 不起新任务 |
| **12:00 - 13:30** | 午饭 + 休息 | 不工作 |
| **13:30 - 15:30** | 📞 中能耗：客户拜访 / 会议 / 沟通 / 报价跟进 | 不写长文档 |
| **15:30 - 17:00** | 🧹 低能耗：整理 / 归档 / 流程（不含供应商入库） | 不做决策 |
| **17:00 - 17:30** | 复盘今日主任务、写明日3件事 | 不新开活 |

---

## 每日三问

| 时间 | 问题 |
|------|------|
| 到岗 | 今天哪件事干完了就可以安心下班？ |
| 午休 | 上午的深度时间用对了吗？ |
| 下班前 | 今天必须做完的做完了吗？没做完的明天什么时候做？ |

## 每周节奏

| 星期 | 主题 | 说明 |
|------|------|------|
| 周一 | 规划日 | 排本周客户优先级、锁定3个主任务 |
| 周二三 | 进攻日 | 保护深度时间，不约内部会 |
| 周四 | 跟进日 | 催回款、跟交付、走流程 |
| 周五 | 复盘日 | 本周完成情况 + 下周预览 + 知识沉淀 |

## 单兵作战守则

没有可委派的人力，所以比时间管理更重要的是**不做清单**：

### 坚决不做
- 在公司内部系统上花超过 30 分钟（录库、填表）
- 参加没有你发言环节的内部会议
- 亲自跟进一切（只跟大客户决策链）

### 能砍就砍
- 售后/交付/巡检 — 找对应部门，不自己做接口人
- 标书排版/格式 — 找商务支持，你只写技术应答内容
- 常规客户拜访 — 月度1次集中跑，不零散约

### 领导层对话
建议找老板谈一次：
- "大企业赛道只有我一个人，我需要一个商务助理或者交付接口人"
- 最轻的方案：让金融组的同事在你出差或紧急时临时支援
