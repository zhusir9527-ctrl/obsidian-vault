# 云易捷 销售红宝书 V1.2（学习笔记）

> 源文件：[QingCloud Express销售红宝书V1.2new.docx](file:///Users/zhu/Desktop/青云网盘镜像/云易捷/销售红宝书V1.2.docx)

## 产品定位

QingCloud Express 是青云企业云平台的轻量版，配合青立方超融合，开箱即用。

三大价值：极简（一键安装扩容）/ 完整（计算+存储+网络+管理）/ 可扩展（→标准版→高级版→企业版）

## 五大需求场景

| 场景 | 目标 | 方案优势 |
|------|------|---------|
| 中小企业云平台 | 政府/教育/金融/医疗 | 一体化交付，公有云验证 |
| 分支机构云平台 | 集团/连锁/交通/教育 | 子域部署统一管理，VMware纳管 |
| 桌面云平台 | 全行业办公 | 秒级启动，低延迟，多种协议 |
| 开发测试云 | 研发部门 | 自助服务，多租户隔离，分时处理 |
| 容灾云平台 | 全行业 | 分钟级RPO/RTO，业务级接管 |

## 竞品打击策略

| 竞品 | 打击要点 |
|------|---------|
| **VMware** | 性价比高、开放性（不被锁定）、自主可控 |
| **Nutanix** | 方案完整（含网络/安全）、自主可控、开放性 |
| **华为** | 网络服务优势、方案完整、灵活开放（不绑硬件） |
| **华三** | 架构精简、管理便捷、开放性 |
| **深信服** | 方案完整、技术成熟度、交互友好 |
| **联想** | 自主可控核心技术（非OEM） |
| **ZStack** | 低起点上云、自助运维、自主可控 |
| **开源OpenStack** | 自主可控、技术成熟、易用（无需自运维） |

## 关键话术

**商机判断**：预算20-30万/6台以下服务器 → 易捷版

**报价策略**：纯软件利润高，软硬一体金额大
