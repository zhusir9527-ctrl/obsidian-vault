## 定价模式

两种模式：

| 模式 | 计费粒度 | 适用场景 |
|------|---------|---------|
| License买断 | 每GPU/每CPU 一次性 | 企业私有化部署、长期使用 |
| 订阅制 | 每GPU/年 或 每CPU/年 | 预算灵活、按年采购 |

## 典型场景组合

### 场景1：企业内部部署大模型API（如DeepSeek）

```
【必选】基础平台 + GPU计算模块 + 分布式推理模块 + 大模型MaaS服务模块
【推荐】GPU池化模块 + 运营模块-计量模块 + 并行文件存储服务模块
【不选】分布式训练模块、HPC系列、财务模块
```

列表价：87,040（核心4模块，8GPU）
全套价：132,640（7模块，8GPU）
参考折扣：4折

### 场景2：算法团队训练专有模型

```
【必选】基础平台 + GPU计算模块 + 分布式训练模块
【推荐】分布式推理模块 + 并行文件存储服务模块 + GPU池化模块
```

### 场景3：算力中心对外运营

```
【必选】全套+运营模块(计量+财务)
```

## 定价逻辑要点

1. **所有模块最低8卡起订**（除CPU模块按CPU计）
2. **基础平台不单独售卖**，必须叠加业务模块
3. **大模型MaaS模块必须联合购买分布式推理模块**
4. **HPC与AI模块独立计费**，同时需要则分别按节点规模下单
5. **实施服务必选**（10人天起），含部署+培训
6. **首年维保免费**，续费按列表价10%/年
7. **订阅制约为买断制的1/3（年付）**，约3年回本
