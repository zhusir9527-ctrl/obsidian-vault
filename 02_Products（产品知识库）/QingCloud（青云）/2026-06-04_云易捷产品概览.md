# 云易捷产品概览

## 产品定位

云易捷（CloudEasy）是青云科技专为**中小规模算力**打造的**轻量化私有云产品**，属于青云私有云产品线的核心组成部分。产品以"从简出发，融合就绪"为理念，提供超融合一体化解决方案，帮助企业快速构建私有云基础设施。

云易捷同时提供**信创版本**（信创云易捷），定位为安全可控、轻量高效的信创超融合产品，适配国产化软硬件生态。

与云易捷配套的青云还提供**信创虚拟化**软件，定位为安全可靠、稳定易用的国产虚拟化软件，可作为 VMware 替代方案的核心组件。

## 核心功能

### 1. 虚拟化能力
- 计算虚拟化：支持虚拟机全生命周期管理
- 存储虚拟化：整合 NeonSAN 分布式块存储和 U10000 非结构化存储
- 网络虚拟化：VPC、负载均衡、安全组、弹性公网 IP
- **KubeVirt 集成**：支持在 Kubernetes 上管理和运行虚拟机，实现容器与虚拟机统一管理

### 2. 超融合架构
- 计算、存储、网络深度融合
- 去中心化分布式架构
- 弹性扩展，按需扩容节点
- 统一管理界面

### 3. VMware 替代方案
青云科技明确提出**国产虚拟化 VMware 替代**解决方案，核心能力包括：
- **信创虚拟化软件**：对标 VMware vSphere 的核心虚拟化能力
- 虚拟机无缝迁移（VMware → 青云虚拟化）
- 兼容现有 VMware 运维流程和工具链
- 支持多种国产 CPU 架构（飞腾、鲲鹏、海光、兆芯等）

### 4. 轻量化部署
- 最小 3 节点起步
- 在线/离线安装
- 快速部署，小时级上线
- 简易运维，图形化管理

### 5. 存储能力
- 对接 **NeonSAN 分布式块存储**：全闪架构、高性能、稳定可靠
- 对接 **U10000 非结构化存储**：海量数据统一存储
- 支持存储快照、备份、容灾

### 6. 信创全栈适配
信创云易捷额外提供：
- 适配国产 CPU：飞腾、鲲鹏、海光、兆芯、龙芯等
- 适配国产操作系统：麒麟、统信 UOS 等
- 适配国产数据库：达梦、人大金仓、南大通用等
- 适配国产中间件：东方通、宝兰德等
- 一云多生态，全栈全场景

## 技术架构

### 整体架构

```
┌─────────────────────────────────────┐
│          云易捷 管理平台              │
│    (统一管理界面 / REST API)          │
├─────────────────────────────────────┤
│  计算虚拟化  │ 存储虚拟化 │ 网络虚拟化 │
│  (KVM/     │ (NeonSAN/ │ (SDN/    │
│   KubeVirt)│  U10000)  │  VPC)    │
├─────────────────────────────────────┤
│      分布式超融合基础设施层            │
│  (x86 / ARM 混合架构服务器集群)        │
└─────────────────────────────────────┘
```

### 关键技术特性
- 基于 KVM 虚拟化技术
- 支持 KubeVirt 容器化虚拟机管理
- 分布式存储引擎（NeonSAN 全闪架构）
- 软件定义网络（SDN）
- 多租户隔离
- 高可用与灾备

## 竞争优势

1. **轻量化定位**：专为中小规模设计，降低私有云门槛
2. **超融合一体化**：计算+存储+网络深度融合，简化架构
3. **从简出发**：快速部署、简易运维、低成本起步
4. **VMware 替代**：明确的国产虚拟化替代方案，降低 license 成本
5. **信创全栈**：完整国产化生态适配，满足合规要求
6. **青云生态协同**：与 KubeSphere 容器平台、AI 智算平台无缝集成
7. **平滑演进**：从虚拟化到容器化（KubeVirt → KubeSphere）平滑过渡
8. **企业级存储**：NeonSAN 全闪架构，高性能可靠存储
9. **一体机交付**：软硬件一体化，开箱即用，降低集成复杂度

## 适用场景

- **中小企业私有云建设**：低成本构建私有云基础设施
- **VMware 替代与国产化**：响应信创政策，替换 VMware 虚拟化平台
- **边缘/分支站点**：轻量化部署，适配边缘计算场景
- **开发测试环境**：快速构建开发测试云平台
- **虚拟化+容器混合场景**：通过 KubeVirt 实现传统 VM 与容器统一管理
- **数据中心整合**：超融合架构整合计算与存储资源
- **信创合规场景**：政务、金融、电力、电信等关键领域国产化部署
