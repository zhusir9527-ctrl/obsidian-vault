## 1. Obsidian MCP配置

### 概述
Obsidian MCP服务通过本地HTTP服务暴露Obsidian Vault的读写能力，使AI Agent可以直接操作知识库中的笔记。

### 功能清单
| 功能 | 方法 | 说明 |
|------|------|------|
| 文件列表 | `list_files_in_vault` / `list_files_in_dir` | 浏览知识库目录结构 |
| 文件读取 | `get_file_contents` / `batch_get_file_contents` | 读取笔记内容 |
| 文件写入 | `append_content` | 追加内容到文件 |
| 文件更新 | `patch_content` | 定向修改文件内容 |
| 文件删除 | `delete_file` | 删除文件或目录 |
| 内容搜索 | `simple_search` / `complex_search` | 全文/结构化搜索 |
| 周期性笔记 | `get_periodic_note` / `get_recent_periodic_notes` | 日/周/月/季/年笔记 |
| 变更追踪 | `get_recent_changes` | 最近修改的文件列表 |

### 配置文件（Claude Desktop / Hermes Agent）

```json
{
  "mcpServers": {
    "obsidian": {
      "command": "node",
      "args": [
        "/path/to/obsidian-mcp-server/dist/index.js"
      ],
      "env": {
        "VAULT_PATH": "/Users/{{username}}/Documents/Obsidian Vault",
        "OBSIDIAN_API_KEY": "{{your_api_key}}"
      }
    }
  }
}
```

### 环境变量说明
| 变量 | 必填 | 说明 |
|------|------|------|
| `VAULT_PATH` | 是 | Obsidian知识库的本地路径 |
| `OBSIDIAN_API_KEY` | 是 | Obsidian Local REST API密钥 |

### 安装步骤
1. 在Obsidian中安装并启用 **Local REST API** 社区插件
2. 在插件设置中生成API Key（非安全模式）
3. 配置MCP Server指向Obsidian MCP服务
4. 启动MCP Server，验证连接：
   ```
   obsidian_list_files_in_vault → 应返回知识库目录
   ```

### 常用操作示例

**搜索客户相关笔记：**
```
obsidian_simple_search(query="XX银行")
```

**创建客户档案：**
```
obsidian_append_content(
  filepath="03_Customers（客户知识库）/XX银行/客户档案_2026-06-04.md",
  content="..."
)
```

**获取今日笔记：**
```
obsidian_get_periodic_note(period="daily")
```

## 2. 微信Gateway配置

### 概述
微信Gateway通过MCP协议桥接企业微信/个人微信，实现消息收发、客户触达、通知推送等功能。用于售前场景中的客户沟通、会议提醒和协作通知。

### 功能清单
| 功能 | 说明 | 售前场景 |
|------|------|----------|
| 消息发送 | 向指定联系人或群发送消息 | 发送方案摘要、会议提醒 |
| 消息接收 | 监听并解析收到的消息 | 捕获客户反馈、需求变更 |
| 文件传输 | 发送文件（PDF、图片等） | 发送方案书、白皮书 |
| 联系人查询 | 搜索企业通讯录 | 查找客户联系人信息 |
| 群管理 | 群创建、成员管理 | 项目协作群管理 |
| 通知推送 | 系统事件通知 | 竞品动态告警、投标提醒 |

### 配置文件

```json
{
  "mcpServers": {
    "wechat-gateway": {
      "command": "python",
      "args": [
        "-m",
        "wechat_mcp_gateway.server"
      ],
      "env": {
        "WECHAT_TYPE": "{{wecom | personal}}",
        "WECHAT_CORP_ID": "{{your_corp_id}}",
        "WECHAT_CORP_SECRET": "{{your_corp_secret}}",
        "WECHAT_AGENT_ID": "{{your_agent_id}}",
        "WECHAT_WEBHOOK_URL": "{{your_webhook_url}}",
        "GATEWAY_PORT": "8765"
      }
    }
  }
}
```

### 环境变量说明
| 变量 | 必填 | 说明 |
|------|------|------|
| `WECHAT_TYPE` | 是 | `wecom`（企业微信）或 `personal`（个人微信） |
| `WECHAT_CORP_ID` | 条件 | 企业微信CorpID |
| `WECHAT_CORP_SECRET` | 条件 | 企业微信应用Secret |
| `WECHAT_AGENT_ID` | 条件 | 企业微信应用AgentID |
| `WECHAT_WEBHOOK_URL` | 否 | Webhook回调地址 |
| `GATEWAY_PORT` | 否 | 网关服务端口，默认8765 |

### 安装步骤
1. **企业微信：**
   - 登录企业微信管理后台
   - 创建自建应用，获取 CorpID、AgentID、Secret
   - 配置应用可见范围和API权限
   - 设置接收消息服务器URL

2. **安装Gateway：**
   ```bash
   pip install wechat-mcp-gateway
   ```

3. **配置并启动：**
   ```bash
   python -m wechat_mcp_gateway.server
   ```

4. **验证连接：**
   发送测试消息到指定用户，确认送达。

### 售前场景自动化

**客户拜访提醒：**
```yaml
触发：拜访前30分钟
```

## 3. 搜索工具配置

### 概述
搜索工具MCP为Agent提供互联网信息检索能力，用于客户研究、行业分析、竞品情报采集等场景。

### 功能清单
| 功能 | 说明 | 售前场景 |
|------|------|------|
| 网页搜索 | 通用网页搜索 | 客户背景调研、行业动态 |
| 新闻搜索 | 新闻资讯检索 | 客户最新动态、行业政策 |
| 企业信息 | 工商/企业信息查询 | 客户企业信息核实 |
| 技术文档 | 技术文档与论文检索 | 技术方案参考、竞品技术分析 |
| 招投标信息 | 招标公告检索 | 商机发现、竞品投标动态 |

### 配置文件

```json
{
  "mcpServers": {
    "web-search": {
      "command": "node",
      "args": [
        "/path/to/search-mcp-server/dist/index.js"
      ],
      "env": {
        "SEARCH_PROVIDER": "{{brave | tavily | serper}}",
        "SEARCH_API_KEY": "{{your_search_api_key}}",
        "MAX_RESULTS": "10",
        "SEARCH_REGION": "cn",
        "SEARCH_LANGUAGE": "zh-CN"
      }
    }
  }
}
```

### 支持的搜索提供商

| 提供商 | 特点 | 推荐场景 |
|--------|------|----------|
| **Brave Search** | 独立索引、隐私友好 | 通用搜索 |
| **Tavily** | AI优化搜索、结构化输出 | 客户研究、深度分析 |
| **Serper** | Google搜索API | 全面覆盖 |
| **Bing API** | 微软生态 | 企业场景 |

### 环境变量说明
| 变量 | 必填 | 说明 |
|------|------|------|
| `SEARCH_PROVIDER` | 是 | 搜索提供商选择 |
| `SEARCH_API_KEY` | 是 | 对应提供商的API Key |
| `MAX_RESULTS` | 否 | 单次搜索最大结果数，默认10 |
| `SEARCH_REGION` | 否 | 搜索区域，默认`cn` |
| `SEARCH_LANGUAGE` | 否 | 搜索语言偏好，默认`zh-CN` |

### 安装步骤
1. 注册搜索服务获取API Key：
   - Brave Search: https://brave.com/search/api/
   - Tavily: https://tavily.com/
   - Serper: https://serper.dev/

2. 安装MCP Server：
   ```bash
   npm install -g @anthropic/search-mcp-server
   ```

3. 配置环境变量并启动

4. 验证搜索功能：
   ```
   web_search(query="XX公司 数字化转型 2026")
   ```

### Agent集成示例

**AccountResearch中的搜索调用：**
```python
# 客户基础信息搜索
results = search_web(
    query=f"{customer_name} 企业 营收 员工",
    max_results=5

    query=f"{customer_name} {industry} 排名 市场份额",

# 技术栈搜索
tech_results = search_web(
    query=f"{customer_name} 技术架构 IT系统",
    max_results=5
)

# 最新动态搜索
news_results = search_news(
    query=f"{customer_name}",
    time_range="90d",
    max_results=10
)
```

```python
    query=f"{customer_name} {competitor_name} 合作 签约",
```

## 5. 使用说明

### 快速验证
启动所有MCP服务后，执行以下验证步骤：

1. **Obsidian连接验证：**
   ```
   obsidian_list_files_in_vault → 应返回目录列表
   ```

2. **微信连接验证：**
   ```
   wechat_send_message(to="self", content="MCP连接测试成功")
   ```

3. **搜索连接验证：**
   ```
   web_search(query="企业数字化转型 趋势 2026")
   ```

### 故障排查

| 问题 | 可能原因 | 解决方案 |
|------|----------|----------|
| Obsidian无法连接 | API插件未启动 | 检查Obsidian中Local REST API插件状态 |
| Obsidian无法连接 | API Key不匹配 | 重新生成API Key并更新配置 |
| 微信发送失败 | CorpID/Secret错误 | 检查企业微信管理后台凭据 |
| 微信发送失败 | 网络限制 | 确认服务器可访问企业微信API |
| 搜索无结果 | API Key失效 | 检查搜索提供商的API配额 |
| 搜索无结果 | 区域限制 | 调整SEARCH_REGION参数 |

### 安全建议
- 所有API Key使用环境变量，不硬编码在配置文件中
- 微信Gateway建议部署在内网环境
- 定期轮换API Key
- 搜索工具注意API调用频率限制
