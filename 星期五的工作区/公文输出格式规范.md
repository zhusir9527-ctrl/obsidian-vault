# 公文输出格式规范

依据《党政机关公文格式》GB/T 9704-2012

## 页面设置
| 项目 | 标准 |
|------|------|
| 纸张 | A4 |
| 上边距 | 3.7cm |
| 下边距 | 3.5cm |
| 左边距 | 2.8cm |
| 右边距 | 2.6cm |

## 字体字号
| 内容 | 字体 | 字号 |
|------|------|------|
| 标题 | 方正小标宋（或黑体） | 二号 |
| 一级标题 | 黑体 | 三号 |
| 二级标题 | 楷体 | 三号 |
| 正文 | 仿宋 | 三号 |

## 段落格式
| 项目 | 标准 |
|------|------|
| 行距 | 固定值 28磅 |
| 缩进 | 首行缩进 2字符 |
| 段间距 | 段前0行，段后0行 |

## 页码
居中，格式为“— 1 —”
