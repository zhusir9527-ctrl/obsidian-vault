# 公文输出格式规范

## 页面设置（GB/T 9704-2012）
| 项目 | 标准 |
|------|------|
| 纸张 | A4（21×29.7cm） |
| 上边距 | 37mm |
| 下边距 | 35mm |
| 左边距 | 28mm |
| 右边距 | 26mm |
| 行距 | 固定值28磅 |

## 字体字号
| 要素 | 字体 | 字号 | 样式 |
|------|------|------|------|
| 发文机关 | 黑体 | 二号（22pt） | 居中加粗 |
| 标题 | 方正小标宋/黑体 | 二号（22pt） | 居中加粗 |
| 文号 | 仿宋 | 三号（16pt） | 居中 |
| 一级标题 | 黑体 | 三号（16pt） | 加粗 |
| 二级标题 | 楷体 | 三号（16pt） | — |
| 正文 | 仿宋_GB2312 | 三号（16pt） | 首行缩进2字符 |
| 落款日期 | 仿宋 | 三号（16pt） | 右对齐 |

## 文件命名规范
`YYYY-MM-DD_主题_文种.docx`（如：`2026-06-12_投标整改方案.docx`）
