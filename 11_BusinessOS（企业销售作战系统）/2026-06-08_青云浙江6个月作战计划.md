# 青云浙江 — 6个月作战计划（2026.06-2026.11）

> 约束条件：2人销售团队 + 后台支撑弱 + 品牌OK
> 核心策略：收缩聚焦，只做三类客户、三个产品包、一个圈子

## 月度主题

| 月 | 主题 | 核心任务 |
|---|------|---------|
| 6月 | 基建月 | 标准化武器库、客户分级名单 |
| 7月 | 出击月 | 密集拜访、圈子启动 |
| 8月 | 攻坚月 | 卡位Q3关键项目 |
| 9月 | 临门一脚 | 冲刺Q3签单 |
| 10月 | 交付月 | 集中交付 + 催续保 |
| 11月 | 收官月 | 全年收官 + 明年布局 |

## 两个销售分工

- **你（老销售）**：攻坚大客户（制造）+ 做圈子KOL + 写售前方案
- **另一销售**：扫中小客户（VMware替代）+ 跟进线索 + 存量维护

## 6个月核心KPI

| 指标 | 目标值 |
|------|--------|
| 签单数 | ≥ 3单（保守），≥ 5单（理想） |
| 金牌客户覆盖 | ≥ 5家深度跟进 |
| 圈子/微信群 | ≥ 80人 |
| Meetup | ≥ 4期 |
| 标准方案包 | 3套，迭代 ≥ 2次 |
| 续保 | ≥ 1单 |

## 三大产品包

1. VMware替代包：信创虚拟化 + 云易捷
2. 容器平台包：KubeSphere + KubeKey + 培训
3. AI智算包：KubeSphere + GPU纳管

## 定价铁律

- 标准折扣 ≤ 15%
- 报价低于标准价30%直接放弃
- 烂单不跟，精力比什么都贵
