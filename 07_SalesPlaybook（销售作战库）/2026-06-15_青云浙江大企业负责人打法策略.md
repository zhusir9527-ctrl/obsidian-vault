# 青云浙江大企业负责人打法策略

> 岗位：青云科技售前总监 | 赛道：制造业 | 区域：浙江 | 团队：2人

---

## 一、浙江战场态势

### 1.1 基本盘判断

| 维度 | 现实 | 含义 |
|------|------|------|
| 阿里云 | 杭州总部，浙江主场 | 正面拼价格拼生态必死，必须差异化 |
| 华为云 | 浙江政企关系极深 | 信创/政务项目华为天然优势 |
| 青云品牌 | 浙江认知度弱于一线城市 | 需要「案例势能」打开局面 |
| 团队 | 2人 | 不能撒胡椒面，必须聚焦打穿 |

**核心结论：浙江不能做「面」只能做「点」。聚焦 3-5 个行业、10 个以内灯塔客户打深打透。**

### 1.2 浙江优势产业（按青云产品匹配度排序）

| 优先级 | 行业 | 代表企业 | 青云切入点 | 竞争态势 |
|--------|------|---------|-----------|---------|
| ⭐⭐⭐ | 纺织化纤 | 恒逸/荣盛/桐昆/华孚 | AI智算+容器平台 | 阿里云强但行业理解弱 |
| ⭐⭐⭐ | 半导体 | 士兰微/立昂微/中芯宁波 | 私有化+K8s+AI训练 | 华为强，但非标需求多 |
| ⭐⭐⭐ | 汽车零部件 | 均胜/拓普/万丰 | 混合云+边缘+DevOps | 格局未定，窗口期 |
| ⭐⭐ | 化工新材料 | 巨化/新安/传化 | 工业互联网+私有云 | 较传统，信创需求 |
| ⭐⭐ | 装备制造 | 三花/海天/正泰 | 容器化改造+AI | 阿里云已渗透 |
| ⭐ | 家电 | 方太/公牛/奥克斯 | 多云管理+DevOps | 腾讯云优势 |

## 二、产品打法：三把刀

### 刀一：KubeSphere 容器平台 — 撕开口子

**定位**：不是卖容器，是卖「不被锁定」的能力。

**目标客户画像**：
- 已有VMware/OpenShift但成本过高的
- 正在做「去IOE」/信创替换的
- 多云战略但缺统一管理面的
- IT团队10人以上有能力自运维的

**竞争话术（vs阿里云ACK）**：
> "ACK是阿里云生态的入口，进去容易出来难。KubeSphere一套配置管私有云+阿里云+边缘端，真混合多云。"

**竞争话术（vs华为云CCE）**：
> "华为全栈信创很好，但您要考虑——三年后如果要换硬件品牌或者上公有云，GaussDB/CCE的迁移成本是多少？KubeSphere 100% CNCF开源标准，不存在锁定。"

**竞争话术（vs灵雀云/谐云等本土厂商）**：
> "灵雀云团队不到100人，青云300+人研发，CNCF贡献中国前三。选平台要看5年，不是看报价单。"

### 刀二：AI 智算平台 — 抬高客单价

**定位**：容器是入口，AI是扩单。KubeSphere拿下后，「你们GPU利用率现在多少？」就是下一步。

**打法**：
- 容器项目（50-100万）→ 6个月后推AI平台（200-500万）
- 振石就是典型路径：先K8s投标→再AI基础设施投标

**核心卖点**：
- GPU利用率从30%→70%（MIG+时分复用+弹性调度）
- 训练+推理+模型管理一个平台
- 适配昇腾/海光等国产GPU（信创刚需）

### 刀三：云易捷一体机 — 降低决策门槛

**定位**：不想谈架构就谈盒子。开箱即用，2周上线。

**话术**：
> "不用讨论部署方案，一个机柜拉过来插电就能用。先跑起来，不满意随时退。"

**适用场景**：
- 制造企业IT能力弱，不想折腾
- 边缘/工厂端轻量化部署
- 信创合规（国产化一体机）

## 三、2人团队杠杆打法

### 3.1 时间分配铁律

| 活动 | 占比 | 说明 |
|------|------|------|
| 拜访决策人（CXO/VP） | 40% | 只约能拍板的人 |
| 方案输出+投标 | 30% | 用AI辅助提效（已有Hermes） |
| 存量客户深耕（扩单） | 20% | 华孚→扩周边纺织客户 |
| 渠道/生态拓展 | 10% | 借ISV/SI的销售团队 |

**禁做清单**：
- ❌ 不做纯技术交流（没有决策人在场的技术交流=无效劳动）
- ❌ 不追200万以下纯盒子单子（代理商做）
- ❌ 不参加无效应酬（每人每周不超过2个饭局）
- ❌ 不做自己都不信能赢的单子（参考振石教训：PoC做了还是输）

### 3.2 借力杠杆

| 杠杆 | 具体操作 |
|------|---------|
| **AI售前助手** | Hermes Agent已可自动生成方案文档、竞品分析、标书偏离表——把30%时间省出来拜访客户 |
| **总部专家远程** | 重要方案评审拉总部架构师远程站台，电话拨入即可，不要求出差 |
| **代理商售前** | 培养1-2家核心代理商的售前能力，标准化方案让他们自己配 |
| **客户Champion** | 每个客户内部培养1个技术KOL，给他CKA培训名额，让他帮你推 |
| **行业案例复利** | 华孚案例→打纺织行业；振石案例→打制造行业；一份方案反复用 |

### 3.3 拜访节奏

```
周一：内部（方案准备+投标输出+情报扫描）
周二~周四：客户拜访（每天最多2家，含路程）
周五：内部复盘+下周规划+行政
```

每周底线：**见够3个决策人或潜在决策人**。

## 四、在跟项目加速策略

### 4.1 华孚时尚（AI智算PoC）

| 维度 | 现状 | 加速动作 |
|------|------|---------|
| 决策人 | 孙伟挺（决策），徐总（CIO） | 每周一次技术推送（不催单，展示专业度） |
| 竞争 | 阿里云/华为云 | 强调「不绑定硬件」+「Token运营计费」差异化 |
| 僵局点 | PoC推进节奏慢 | 申请总部专家远程诊断一次，给一份PoC优化建议书 |
| 扩单机会 | 纺织行业标杆 | 主动提「华孚+青云联合白皮书」——帮他们做行业影响力 |

### 4.2 士兰微（待拜访）

| 维度 | 动作 |
|------|------|
| 拜访前 | 搞清：①当前芯片设计仿真用什么平台？②GPU用哪些？③数据安全合规要求？ |
| 首次见面 | 不带方案，带「半导体行业AI基础设施调研报告」→ 展示行业理解 |
| 关键人 | 找到设计部门/IT部门的分管VP，不是只跟IT工程师聊 |
| 切入策略 | 半导体设计仿真（EDA上云+GPU调度）作为切入点 |

### 4.3 振石（K8s已丢，AI投标中）

| 维度 | 动作 |
|------|------|
| K8s教训 | 输给灵雀云→复盘报告已写，关键教训：PoC准备不充分+未渗透到决策层 |
| AI投标 | 重点打「开源开放不锁定」+「已与主流硬件适配」 |
| 关系修复 | 即使K8s输了，继续保持技术交流，AI翻盘+后续运维工具机会 |

## 五、客户分层与拓展路径

### 5.1 三层漏斗

```
第一层（在跟）：华孚、士兰微、振石AI
    ↓
第二层（培育）：恒逸/荣盛（纺织）、均胜（汽车）、巨化（化工）
    ↓
第三层（扫描）：浙江制造百强中有容器/AI需求的
```

### 5.2 获客三板斧

| 手段 | 操作 | 预期效果 |
|------|------|---------|
| **行业白皮书** | 出一份「浙江制造业AI算力白皮书」青云署名 | 品牌占位+敲门砖 |
| **老客户转介绍** | 华孚案例→引荐桐昆/新凤鸣 | 纺织行业链式渗透 |
| **代理商地推** | 找到1-2家在浙江制造业有资源的ISV/SI | 借鸡生蛋 |

### 5.3 关键KPI（季度）

| 指标 | 目标 |
|------|------|
| 拜访决策人 | ≥36人次/季度（每周3人） |
| 新客户触达 | ≥6家/季度 |
| 方案/投标输出 | ≥4份/季度 |
| 中标金额 | ≥300万/季度 |
| 案例沉淀 | ≥2个可对外讲的案例/季度 |

## 六、竞争速查卡

### 遇到阿里云时：

| 我方打 | 对方弱点 | 话术 |
|--------|---------|------|
| 不锁定 | 进去容易出来难 | "ACK深度绑定阿里云生态，三年后迁移成本您算过吗" |
| 混合多云 | 只能管自己 | "您在阿里云上的GPU怎么跟本地训练集群统一调度？" |
| 数据主权 | 数据必须上云 | "制造企业核心工艺数据放公有云，您安全部门同意吗" |
| 私有化 | 公有云为主 | "断网演练过吗？工厂内网环境ACK怎么跑？" |

### 遇到华为云时：

| 我方打 | 对方弱点 | 话术 |
|--------|---------|------|
| 开源开放 | 华为自研技术栈 | "GaussDB+CCE+昇腾绑定太深，万一芯片供应出问题？" |
| 多芯片适配 | 昇腾锁定 | "我们适配昇腾/海光/寒武纪/英伟达，您不用押宝单一芯片" |
| 性价比 | 华为价格偏高 | "同等规模华为报价通常高30-50%，开源方案运维成本低" |
| 轻量灵活 | 方案偏重 | "华为最小部署也要5节点起，我们3节点就能跑" |

### 遇到本土厂商（灵雀云/谐云/才云等）时：

| 我方打 | 对方弱点 | 话术 |
|--------|---------|------|
| 公司体量 | 团队规模小 | "百人团队vs三百人研发，CNCF贡献中国前三" |
| 全栈能力 | 只有容器 | "他们有AI平台吗？有分布式存储吗？有一体机吗？" |
| 长期服务 | 融资型公司 | "上周又裁员了？您的平台五年后还有人维护吗" |
| 信创生态 | 生态不全 | "我们已完成20+国产芯片/OS/数据库互认证" |

## 七、每周作战节奏

```
周一上午：情报扫描（产品动态+竞品动向+政策变化）
周一下午：方案输出+投标推进（华孚/振石AI/新项目）
周二：客户拜访A（上午一家，下午一家）
周三：客户拜访B（上午一家，下午一家）
周四：客户拜访C + 渠道沟通
周五上午：内部复盘（进度/问题/下周计划）
周五下午：方案收尾+行政
```

**每周五发一条微信给3个关键客户/潜在客户**（行业动态/技术洞察/案例分享，不推销）。
