| **深绿** | `#00AA72` | 渐变起点 |
| **中间绿** | `#24B77F` | 渐变中段 |
| **浅绿** | `#86DBA2` | 渐变终点 |

### 辅助色（AI/科技线）

### 中性色
| 用途 | 色值 |
|------|------|

## 二、渐变风格

青云 PPT 的最大特征是**渐变**替代传统色块:

1. **品牌渐变**(水平 90deg): `#00AA72 → #24B77F → #86DBA2`
   - 用于封面、章节页、标题栏、CTA 按钮
   
2. **科技渐变**(92deg): `#00C688 → #28A9FF → #1977FF`
   - 用于 AI/智算/云原生主题页面
   - 体现"从云到 AI"的技术演进

## 四、设计风格特征

1. **极简科技风**: 大量留白，信息密度适中，不堆砌
2. **渐变即装饰**: 不用传统图标/插图背景，用品牌渐变条做视觉锚点
3. **卡片式布局**: 产品/方案用圆角卡片排列，hover 有微妙阴影
4. **中英双语标签**: 每个产品/方案有中英文双行标题(如"公有云 PUBLIC CLOUD")
5. **数据可视化**: 大数字 + 简短标签(如"18000+ 企业用户")，绿色数字
6. **封面公式**: 左侧品牌渐变装饰条 + 大标题 + 副标题(中英文) + 日期/演讲人

## 五、页面类型模板

### 封面页
- 纯白背景/深色背景
- 左侧或顶部品牌渐变条
- 大标题居中或左对齐
- 中英文双行标题
- 底部: 青云 Logo + 股票代码 688316

### 章节分隔页
- 大面积品牌绿色或深色背景
- 居中大数字编号 + 章节标题
- 渐变装饰线

### 内容页
- 白色背景
- 顶部渐变细线(品牌绿)
- 标题左对齐，正文分栏
- 架构图/流程图居中以白色卡片承载

### 数据页
- 大数字 + 渐变底色
- 简洁标签
- 四列/三列网格排列

## 六、Confluence 内部规范补充

从 Confluence 搜索结果:
- 青云有正式的**市场活动指导规范**(含 PPT 视觉标准)
- 有**文档规范 v1.0**(服务团队)
- 架构图有统一规范(可用于 PPT/白皮书/官网)
- 客户案例有固定模板(封面 + 企业介绍 + 挑战 + 方案 + 效果)
- "一云多生态 全栈全场景"是高频 Tagline

## 七、PPT 制作建议

1. **封面**: 白底 + 左侧品牌绿渐变条 + 大标题
2. **配色严格遵守**: 只用品牌绿系 + 科技蓝系，不用第三方颜色
3. **每页标题区**: 顶部细渐变线(2-3px)
4. **产品名称**: 中英双语格式
5. **架构图**: 深色背景 + 品牌绿/蓝渐变节点
6. **数据展示**: 大数字绿色 + 小标签灰色
7. **页脚**: 青云 Logo + 688316 + 页码
